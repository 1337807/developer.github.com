Repository Contributors and Empty Repositories

  • February 20, 2014
  • izuzak

Weâ€™ve made a small change to the Repository Contributors API in the way empty repositories are handled. Previously, the API returned a 404 Not Found status when the list of contributors was fetched for an empty repository.
